# Collaborative Whiteboard

Real-time collaborative canvas. Draw together in the browser, no plugins needed.

---

## 🚀 Deploy to Glitch (free, 5 minutes, no CLI)

1. Зайди на **https://glitch.com** → войди (GitHub или email)
2. **New Project → Import from GitHub** (или шаблон `glitch-hello-node`)
3. Удали дефолтные файлы, загрузи наши `server.js`, `package.json`, `index.html`
   - В редакторе Glitch: ⋮ рядом с файлом → **Upload a file**
4. Glitch сам запустит `npm install && node server.js`
5. **Share** → скопируй URL вида `https://ваш-проект.glitch.me`
6. Открой URL в двух браузерах → одинаковый Room ID → рисуй!

> Бесплатный Glitch засыпает после 5 мин неактивности (первый запрос ждёт ~10с).

---

## 🚀 Deploy to Render (альтернатива)

1. Залей на GitHub
2. render.com → New → Web Service → подключи репо
3. Build: `npm install`, Start: `node server.js`
4. Получи URL → открывай в любом браузере

> Бесплатный Render тоже засыпает. Пинг через UptimeRobot решает.

